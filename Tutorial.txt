Executive steps:

Install the following modules:
* seaborn
    pip install seaborn or conda install seaborn
* fpdf
    pip install fpdf or conda install fpdf
* pyarrow
    pip install pyarrow or conda install pyarrow
* fastparquet
    pip install fastparquet or conda install fastparquet

Procedere con l'analisi dei data sui notebook

Eseguire gli script nelle utilities:
* clean data
* read data
* analyze data
* graph data

Infine, eseguire lo script analyzed_nyc_yt_trip.py:
* Aprire il terminale e posizionarsi nella folder principale all'interno della propria repository:
    ```shell
    foo@bar:~$ cd NY_YellowTaxi_Analytics/
    ```
* Lanciare lo script:
    ```shell
    foo@bar:~$ python3 analyzed_nyc_yt_trip.py
    ```
